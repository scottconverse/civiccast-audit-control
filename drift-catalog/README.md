# Drift Catalog

This directory starts empty. Add one Markdown record per detected mismatch among CivicCast source, coder-side references, canonical verdicts, pull-request state, release state, and audit-control governance.

Each record includes:

- detection date
- affected repository and exact SHA/tag
- observed state and expected state
- reproduction/detection command
- severity and affected gate
- owner and disposition
- closing SHA and verification evidence

An empty directory means no drift has been recorded. It is not proof that no drift exists.
